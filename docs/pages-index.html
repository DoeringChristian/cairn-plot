cairn-plot — live examples

Every page below is ONE self-contained HTML file rendered by CI from the latest main — the same output import cairn_plot as cp produces locally. No server, no CDN; everything (data, renderer bundles, WebGPU kernels) is baked in.

Gallery — every plot type

2D charts, tables, SDR + true-float HDR images, all compare modes (incl. FLIP at 2048×2048), WebGL 3D viewers. Large page (~40 MB — the data is inline).

 Feature report — themed cp.Report

Markdown + components in the report layout: compare/diff kernels, FLIP validated against NVIDIA's official flip-evaluator, HDR-FLIP, synced viewports.

 URL images — client-side EXR decoding

The page bakes no pixels — the browser fetches and decodes OpenEXR (ZIP + PIZ) itself. Needs network when viewed.
GPU features need WebGPU (a notice appears in-page if your browser lacks it) — see the browser support guide. Source: github.com/doeringchristian/cairn-plot.
